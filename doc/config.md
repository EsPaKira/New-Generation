# Конфигурация New Generation

## Урон при удушье

Урон, который наносится, когда заканчивается кислород.

Базовое значение - 10000

Можно изменить значение "suffocation-damage" в newgen:config/newgen.json или создать в вашем паке config/newgen.json с отличным уроном при удушье

### Пример *config/newgen.json*:
```json
{
    "suffocation-damage": 10
}
```

## Фоны интерфейса

Чтобы добавить свой фон, создайте в папке config файл newgen_backgrounds.json. В этом файле должен быть массив, содержащий в себе путь к фону.

Путь к фону выглядит так: **атлас:название_текстуры_без_расширения**, если фон есть в атласах текстур. Если его нету в атласах, то фон должен быть объявлен в **preload.json**; в таком случае путь выглядит так: **папка/название_текстуры_без_расширения**.

### Пример *config/backgrounds.json* из newgen:
```json
[
    "gui/space"
]
```
