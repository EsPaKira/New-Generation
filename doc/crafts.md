Крафты добавляются в папку crafts (либо в контент-паке New Generation, либо в вашем паке).

Если хотите дополнить уже существующие крафты, в папке crafts вашего контент-пака создайте json-файл с тем же названием.

Крафты из primitive_crafts.json доступны везде; остальные крафты доступны в своих верстаках.

*Крафты для печек объявляются только в melting_crafts.json*


Шаблон для json-файла крафтов:

```json
[
    {
        "components": [
            {"id": "ID_контент-пака:ID_предмета", "count": 777}, ...
        ],
        "results": [
            {"id": "ID_контент-пака:ID_предмета", "count": 777}, ...
        ]
    }
]
```

Так же доступны крафты по материалам(сейчас в New Generation всего 2 вида материалов: wood и pebble). в таком случае в components вместо "id": "ID_контент-пака:ID_предмета" пишется "tag": "название_tag", где название_tag - wood, pebble или ваш материал.

Чтобы добавить свой материал, в json-файле предмета добавьте поле "newgen:craft-material": "ваш_материал"
