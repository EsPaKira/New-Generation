# О New Generation

Этот контент-пак задумывался как самостоятельная survival RPG с некоторыми возможностями по модификации. Например, на данный момент уже [возможно](config.md) менять фоны интерфейса и урон при удушье через сторонние контент-паки. Так же через сторонние паки можно добавлять [новые крафты](crafts.md), новых мобов с [собственым лутом](entities_loot.md) и сделать блоки-хранилища с [генерацией вашего лута](blocks_loot.md).

Одной из основных систем New Generation должна быть система сюжета. NewGen должен уметь проигрывать как минимум диалоги с выборами, влияющими на сюжет, а как максимум - уметь воспроизводить катсцены. Такие сюжеты должны добавляться через отдельные контент-паки.

На основе New Generation я хочу воплотить в жизнь свою выдуманную в 7-м классе (4 года назад) вселенную - [Project L.A.R.I.X.](https://github.com/EsPaKira/Project-L.A.R.I.X.). Эта история про магию и путешествия по мирам. Минерва (пока что только её моделька готова) - одна из основных дествующих лиц истории.

Другие основные механики NewGen - боевая система и выживание. На будущее запланировано 100500 обновлений этих систем. Ждите магию, фермерство, стрелковое оружие (например луки, арбалеты и мушкеты) и много боссов.

Так же в ближайшем будущем хочу доделать основу системы персонажей: осталось добавить возможность смены игрового персонажа и систему получения персонажей (вероятно будет гача, но хотелось бы так же добавить API для других способов).
